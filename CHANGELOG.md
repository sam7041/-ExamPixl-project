# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-02

### Added
- **31+ PDF Tools**: Compress, Merge, Split, Rotate, Watermark, Sign, Protect, Unlock, Organize, Page Numbers, Remove/Extract Pages, OCR, JPG/Word/PowerPoint/Excel/HTML to PDF, PDF to Word/PowerPoint/Excel/PDF-A, Scan, Edit, Redact, Compare, Repair, Crop, AI Summarizer, Translate
- **25+ Image Tools**: Compress, Smart Resize, Format Converter, Crop, Remove/Change Background, Rotate & Flip, Filters & Effects, Adjustments, Borders & Padding, Round Corners, Watermark, WebP Optimizer, Strip Metadata, Progressive JPEG, Image to PDF, Batch Process, Collage Maker, Merge, Responsive Generator, OCR Scanner
- **6 Exam-Specific Tools**: Exam Photo Validator, Signature Canvas, Triple Signature, White Background, Declaration Generator, Signature Extractor
- React 19 + TypeScript 5.8 foundation
- Vite 6 build system with optimized code splitting
- Tailwind CSS v4 for responsive design
- Service Worker for offline support and caching
- Google Gemini AI integration (optional) for summarization and translation
- SEO optimization with per-page metadata
- PWA support with manifest and icons
- Deployment configs for Vercel, Netlify, and Cloudflare Pages

### Features
- 100% browser-based processing - no server uploads
- Works offline with Service Worker caching
- Optimized for Indian mobile networks
- Exam-specific photo validation (8+ exams)
- Signature generation and extraction
- All tools run entirely in the browser
- Zero user tracking, completely private

### Performance
- Critical CSS inlined for instant page render
- Code splitting for lazy loading (PDF/Image libraries)
- 1-year cache headers for hashed assets
- ~50KB gzip JS for initial load
- Lighthouse score optimized
